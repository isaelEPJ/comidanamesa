O que é?


Você sabia? Todos os dias são descartados cerca de 41 mil toneladas de alimentos, produtos bons para consumo mas prestes a expirar que vão para o lixo.

Comida na Mesa é um programa que busca levar esses bons alimentos, que seriam descartados por conta do prazo de validade, para a mesa dos cidadãos necessitados de Apucarana - PR. Veja como funciona:

O programa possui basicamente 3 atores: Empresas Alimentícias, Assistência social e Cidadão Necessitado

O papel da Assistência Social é criar uma ponte entre os dois outros atores: organizar os alimentos doados pelas Empresas Alimentícias em kits e distribuir para os Cidadãos Necessitados que fizerem o agendamento para retirada dos mesmos. Para facilitar toda a logistica, o programa possui uma plataforma web com níveis de acesso específicos para cada um desses atores.






Dessa forma todos ganham, além de fazer um ato de caridade, as empresas contam com benefícios fiscais gerados pela doação, o governo ampara sua população e a fome dos cidadãos é amenizada. Quanto maior é a aderência da comunidade, maior é a possibilidade de zerarmos os números de famílias com fome em Aprcarana, vamos participar?
